字符串常见用法

发表于 2017-06-03 | 分类于 Java

字符串相关知识

最近写数据结构和算法的时候经常遇见字符串相关的问问题,有些常见的方法其实java已经内置了函数，应该早就写这篇博客了.
下面将对字符串的常见和一些十分有帮助的函数写下来,从构造函数说起吧。
基本是将JavaApi文档用中文解释一下哈.

字符串相关类

既然提到了String 就不得不提StringBuffer和StringBuilder

  • String字符串常量
  • StringBuffer 字符串变量(线程安全)
  • StringBuilder 字符串变量(非线程安全)

构造函数

  • String()默认空串
  • String(char[] chars) 通过字符数组初始化一个字符串
  • String(byte[] bytes) 通过字节数组初始化
  • String(byte[] bytes,Charset charset) 通过字节数组初始化的同时,指定编码方式
  • String(byte[] bytes,int offset,int length) 通过字符数组初始化的同时,指定起始位置
  • String(byte[] bytes,int offset,int length,Charset charset) 上述基础上指定编码方式
  • String(String original) 通过一个字符串初始化
  • String(StringBuilder original) 通过StringBuilder初始化一个变量
  • String(StringBuffer original) 通过StringBuffer初始化一个变量

常见字符串处理函数

  • char charAt(int index)
  • boolean compareTo(String anotherString)
  • String contact(String str) 连接两个字符串
  • boolean contains(CharSequence s)
  • boolean endWith(String suffex)
  • equalsIgnoreCase(String anotherString)
  • byte[] getBytes()
  • byte[] getBytes(Charset charset)
