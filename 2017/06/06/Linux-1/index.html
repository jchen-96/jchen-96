Linux-1、基本操作之文件管理

发表于 2017-06-06 | 分类于 Linux ， 基本操作

今天linux下的文件基本操作(ubuntu)

写在前面

简单的介绍一下linux命令行的的最前面的几个字符串所代表的意义

Jchen 代表当前登录的用户:root表示超级用户，即管理员，其他为一般用户
localhost 代表当前的主机名称
~ 当前登录用户的home目录
$ 普通用户的提示符
#　超级用户的提示符
命令的基本格式: 命令　【选项】【参数】　个别命令使用不遵循此格式

查询目录中的内容

选项:
-a 显示所有的文件，包括隐藏的文件
-l 显示详细信息
-d 查看目录的属性
-h 个性化显示文件大小
-i 显示inode

文件详细信息的第一个参数是文件类型,-文件、|链接、d目录

目录处理命令

创建目录　mkdir [-p] dirname 加了-p可以递归的创建目录

切换目录 cd [目录]

显示当前的绝对路径 pwd

删除文件　rm -rf [文件或者目录]　-r 删目录　-f 强制删除

复制文件

选项
-r 复制目录
-p 连带文件属性复制
-d r若源文件是链接文件，复制链接属性
-a 相当于 -pdr 原封不动的复制文件的所有属性

剪切或改名命令

linux 常见的目录

/　根目录
/bin 命令保存目录(普通用户就可以读取的命令)
/boot 启动目录。启动相关文件
/dev 设备文件保存目录
/etc 配置文件保存目录
/home 普通用户的家目录
/lib 系统库保存目录
/mnt 系统挂载目录
/media 挂载目录
