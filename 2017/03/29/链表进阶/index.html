Reverse Nodes in K-group(25)

  • 给定一个链表，每k个节点为一组，反转没一组的k个节点。k为正整数且小于链表的长度。如果剩余的链表长度不够K.泽余下部分不需要进行反转。
  • 1->2->3->4->5 k=2 反转操作后 2->1->4->3->5
  • 1->2->3->4->5 k=3 反转操作后 3->2->1->4->5

    给定一个节点删除链表中的该节点(237)

  • 一般链表相关的问题都不能改变链表节点的值，但是有时候必须改变，这个题目只是为了扭转一种思维惯性。
  • 注意是给定要删除的节点，而不是给定删除节点的值。
  • 刚开始可能一脸懵逼，不过可以将下一个节点的值给当前节点，再去删除要删除节点的下一个节点，这样就等价于删除当前节点。
  • 因为删除某一个节点必须要得到删除节点的前一个节点的引用，所以采用这种等价的方式。

    给定一个链表删除倒数第n个节点(19)

  • 举个栗子1->2->3->4->5,n=2 返回 1->2->3->5
  • 设置两个索引当后一个索引到达尾部节点的next(null),保证前一个指针到达要删除节点的前一个节点。
  • 举个栗子画一下比较好，不然容易出错。

    旋转链表(61)

  • 要求：给定一个链表，让这个链表向右边旋转k位。其中k为非负数
  • 举个栗子：1->2->3->4->5,k=2
    • 第一次旋转:5->1->2->3->4
    • 第二次旋转：1->2->3->4->5
  • 返回 1->2->3->4->5

    Reoder List(143)

  • 题目介绍太长，我就懒得翻译
    Given a singly linked list L: L0→L1→…→Ln-1→Ln,
    reorder it to: L0→Ln→L1→Ln-1→L2→Ln-2→…

You must do this in-place without altering the nodes’ values.

For example,
Given {1,2,3,4}, reorder it to {1,4,2,3}
