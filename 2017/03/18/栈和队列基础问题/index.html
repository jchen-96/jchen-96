Valid Parentheses(20)

  • 给定一个字符串，判断字符串中的括号匹配是否合法
  • 举个栗子，”()”,”()[]{}”合法，”(]”,”([)]”不合法。

逆波兰表达式求值(150)

  • 给一个数组求逆波兰表达式的值，也就是后缀表达式的求值。
  • 举个栗子[“2”, “1”, “+”, “3”, ““] -> ((2 + 1) 3) -> 9
  • 再举个栗子[“4”, “13”, “5”, “/“, “+”] -> (4 + (13 / 5)) -> 6

简化路径（71）

  • 给定一个unix下的路径，简化这个路径。
  • 举个栗子，如/home/,简化后的路径为/home
  • 如/a/./b/../../c/,简化后为/
  • 关于unix下的路径表达，自行谷歌
