Servlet执行流程

发表于 2017-04-03 | 分类于 JavaEE

Get方式请求Servlet执行流程

  • GET 方式请求HelloServlet >>>>>>> <a href=”servlet/HelloServlet”>
  • 在web.xml中寻找url-pattern中与之对应的Servlet,得到Servlet-name
  • 在web.xml中寻找满足条件的Servlet(java class)
  • 根据请求方式选择执行post或get方法

Servet生命周期

  • 开始request请求
  • 判断servlet实例是否存在。
    • 存在调用service方法
    • 不存在
      • 装载Servlet类，并创建实例
      • 调用init(ServletConfig)方法
      • 调用service方法
  • 在服务器关闭的时候，调用Destroy方法。

在下列时刻Servlet容器装载Servlet

  • 启动时候自动装载，通过设置1,数字越小优先级别越高。
  • 容器启动后，客户端首次向Servlet发送请求。
  • Servler类文件被更新后，重新装载Servlet。

Servlet与Jsp九大内置对象

Jsp对象 怎样获得

  • out resp.getWriter
  • request service方法中的req参数
  • response service方法中的resp参数
  • session req.getSession()函数
  • application getServletContext()函数
  • execption Throwable
  • page this
  • pageContext pageContext
  • Config getServletCOnfig
