又是一年

发表于 2017-12-31 | 分类于 虾机趴扯

2017就这样渐行渐远了

今年以及大四了明年就毕业了，曾经以为大学四年很长，不知不觉已经大四了，马上就要毕业了。
今年相比于前两年半做的事情还是挺多的。就随便写写，博客又一段时间没有更新了，不是忙，是懒。。。

2017 干了些啥

找实习

由于之前一直是坚持就业的，所以理所当然这学期一开学就开始找实习了。找实习的时候还是考察基础知识，大致可以分为以下几个部分。

  1. 计算机基础主要考察的是计算机网络和操作系统部分，计算机网络一般都是考察三次握手，以及通信系统七层或四层上每一层协议对应的作用，哪个协议位于哪一层不不能说错。操作系统部分一般会问线程和进程的区别和联系。
  2. 数据结构和算法，基础的数据结构包括线性结构，数组、链表、队列、栈。非线性数据结构树、二叉树、搜索二叉树、堆、并查集。基础的算法包括基础排序(O(N2)),高级排序(快排，二路快排、三路快排、堆排序、归并排序)，二分查找。以上提到的都要能用笔撸出来。然后差不多之后，去leetCode 把简单的全撸过，中等难度的把和上面基础数据结构和算法相关的做做就差不多了。
  3. Java基础，面向对象编程，泛型，反射，各种集合的基本使用。集合的源码要看看，特别是hashMap。
  4. JavaEE部分、spring家族那些常见模块的概念和作用。spring框架的核心(AOP和IOC),熟悉web层框架(SpringMVC)的流程(从接收请求到响应的返回),mybatis持久层框架的基本原理和使用。
  5. 数据库的使用(结构性数据库非结构性数据库)，会基本操作(CRDU)
  6. 项目:找一个小项目写一下，跑一下，能独立的整合各个框架进行开发，基本就ojbk了。

一般来说会了上述的东西，除了BAT,一般的互联网公司的实习是没问题的。其实主要的也就是第二部分，面试的时候代码手撸出来了基本就没问题了。

我经验也不是很多，当时就面了一个搜狐，过了。当时总是觉得自己太弱，拿到一个实习offer就直接跑路了。没有面其他的公司，其实完全没有必要，应该多找找。。。。

实习

3月24号拿到sohu的offer后。四月份就直接去北京了。说说实习干了写啥吧。
现在回想起来感觉实习其实也没干多高大上的事情，日常的工作就是维护一个文章的审核平台，理一下代码的流程。
和自己写项目不同的就是可能缓存用的比较多一点还有少许的多线程编程，还有数据量很大看着很刺激。

主要的收获还是工作方面吧，比如如何和同事进行交流。最重要的一点，一定要上班时间认真做事，不管有没有活，没有的话主动一点。当时我就是感觉活不是很多，自己学自己的。让老大感觉我工作不是很上心，导致在秋招的时候技术面都过了，在老板面的时候挂了，也是日了狗了。。。。。。

准备找工作

6月初回来考试之后,7月初考完想了一下，那边技术方面学到的知识很有限。就没去北京了，直接在学校复习秋招的内容。秋招还是比实习难一些，需要准备的知识可能多一点，不过大致还是上述几个部分,上面的肯定是要滚瓜烂熟的，再加一点下面的知识。

  1. 计算机基础部分。计算机网络(http协议1.0,1.1,2.0的异同，简单的网络攻击csrf,xss,tcp拥塞控制)。操作系统(BIO/NIO/AIO)，操作系统(进程通信,os策略，互斥与死锁相关，linux常用命令，linux内核相关)

  2. 算法和数据结构部分，数据结构部分加上图(主要是连个最短路径算法)，把leetCode至少中等撸的很熟.剑指offer这本神书一定要看，多撸几遍。

  3. 数据库(mysql,redis),索引和优化方式，失效条件，底层结构，sql语法(join,union,子查询，having,groupby),数据库的锁，数据库引擎的对比，隔离级别，事务特性，基本优化，分库分表，主从复制，读写分离

  4. Java，并发和多线程，JVM,反射和代理

  5. 项目和实习经历

  6. 高级内容，rpc框架，消息队列，分布式存储系统，分布式事务，分布式锁，脚本语言。这部分只是加分的。觉得上面都ok了，可以学学。

    虽然写起来很少，但是实际内容还是挺多的。比如并发和多线程里面东西很多，还有JVM。

找工作

8月份投的内推简历，基本全都gg。感觉是学历其实，CS大法还是好。。。。。。
9月份开始天天笔试，14号面试了搜狐，boss面挂。。。。

虽然找工作的时间不长，但是找工作是一个磨砺心智的过程，其中的辛苦只有自己能体会。
但是有一点只要你付出了足够多的努力，就会有回报。

突然就读研了

面试完搜狐之后，本来19,20还有滴滴和美团的面试。但是突然听同学说，我可以保研。便抱着试试的心态看看能不能跨去计算机专业，准备了一下导师的面试，通过了。又过了后面的一个笔试，就这样莫名奇妙的读研了。从知道保送到确定读研前后不过一周的时间，之后的面试也都没有去。其实主要还是找工作压力太大，想找个好点的不容易哇，而且自学的和他们本专业的学生，不说实力差很多，但是面试的时候，以及简历的筛选确实比较吃亏。而且既然可以去计算机我还是愿意读个研究生的。

混混

确定了读研之后，工作就不找了，基本从10月份开始就瞎几把浪，什么事也不相干。就偶尔看看导师给的文献和了解一下人工智能相关的领域，想想研究生搞点啥事情。一转眼2017就混过去了。。。。。
这几个月，游戏也打腻，搞得现在什么事都不想干。。。。
说好的总结本科学到的技术也没什么进展。

Next

多看书

大学四年都没怎么提高自己的软实力，导致自己的知识面很窄。
希望自己戒掉那些没有意义的东西，多读书吧。小说，经济，哲学，历史都可以，不要过于功利。

学习还是很重要的

学习搞好了，退路就多了，而且大学的学习其实占不了多少时间。
大二大三的时候一度感觉大一的努力都白白浪费了，但是实际上却成为了保研的关键，虽然后面两年夏季而浪了，但是大一成绩撑着，所以自己还是能顺利保研。所以说嘛，努力从来不会白费。

做事要专注

感觉这方面自己很差劲，老觉得自己可以一下干很多事，但是最终没有一件干的很极致。。。。
需要改改这毛病
努力吧

继续这“无趣”的生活，但自己一定要快乐，用微笑做面纱，像蜗牛一样成长。

现在是2018-12-31 22:00 希望新的一年自己越变越好。。。。今天早点睡
